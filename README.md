# CZM CF Web (Laravel)

## Project Setup

- Install php 8.1, composer, mysql, npm
- run ``composer install``
- run ``npm install``
- run `php artisan key:generate`
- Copy/Create and populate `.env` file

## Run Application

Run these commands in two separate terminal window to run the application in development mode:
````
php artisan serve
````
````
npm run dev
````

## Navigate application

Visit `http://localhost:8000/` or `http://127.0.0.1:8000/`
